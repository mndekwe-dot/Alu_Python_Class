# Alu Python Class

A collection of Python exercises and projects completed during the ALU Python course.

## Projects

### Library Management System ([library.py](library.py))

A simple library system that tracks books and borrow history using OOP and JSON persistence.

- `Book` — stores title, author, year, genre, and borrow status
- `User` — registers users and allows them to borrow books
- `show_history()` — prints the full borrow log
- Data is saved to [library.json](library.json) between runs

**Run:**
```bash
python library.py
```

### Icebreaker ([icebreaker.py](icebreaker.py))

A function that moves zeros to the end of a list while preserving the order of non-zero elements.

**Run:**
```bash
python icebreaker.py
```
